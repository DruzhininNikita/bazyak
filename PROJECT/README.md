# PriceWatcher v2.1 — Мониторинг цен Яндекс.Маркет

## Установка зависимостей

```bash
pip install customtkinter playwright openpyxl plyer Pillow
playwright install chromium
```

## Запуск

```bash
python pricewatcher.py
```

## Возможности

- **Парсинг цен** с Яндекс.Маркета (множество CSS-селекторов + fallback через JSON-LD)
- **Очередь товаров** — мониторьте несколько товаров одновременно
- **Гибкий интервал**: 5 минут / 15 мин / 30 мин / 1 час / 2 часа / 4 часа / 8 часов / 24 часа
- **Уведомления**: Windows (toast/plyer) и Linux (notify-send) и macOS
- **Excel-журнал** (`~/PriceWatcher/price_history.xlsx`) со статистикой
- **Настройки**: тема Dark/Light/System, цветовая схема, смена обоев окна
- **Очистка URL** — убирает трекинговые параметры из ссылки автоматически
- **Смена темы для слабовидящих** — 4 режима доступности в настройках

## Режимы доступности (слабовидящие)

В настройках доступны 4 режима:

| Режим | Описание |
|---|---|
| **Обычная тема** | Standard Dark / Light (по умолчанию) |
| **Высокая контрастность** | Чёрный фон + яркие цвета, максимальный контраст |
| **Режим для дальтоников** | Оттенки синего/оранжевого, различаемые при дейтеранопии |
| **Крупный шрифт** | Увеличенные шрифты (+30%) для всех элементов интерфейса |

Переключение — кнопка «Тема для слабовидящих» в настройках (шестерёнка в шапке).

## Управление приложением

### Основные вкладки

- **Мониторинг** — ввод URL товара, целевой цены и интервала проверки; кнопки запуска/остановки мониторинга; лог событий
- **Очередь** — список всех добавленных товаров с возможностью удаления
- **Статистика** — счётчики проверок, найденных цен, скидок и ошибок текущей сессии

### Кнопка настроек (шестерёнка в шапке)

1. **Тест уведомления** — проверяет работу системных toast-уведомлений
2. **Смена темы для слабовидящих** — переключение между 4 режимами доступности
3. **Выбор обоев окна** — установить фоновое изображение из файла
4. **Сброс обоев** — удалить фоновое изображение

## Примечание по парсингу

Яндекс.Маркет активно использует защиту от ботов. Если цена не находится:
- Попробуйте уменьшить интервал до ≥15 минут
- Убедитесь, что ссылка ведёт прямо на карточку товара (не на список)
- Playwright запускается в режиме stealth (маскируется под обычный браузер)

## Режимы тем для слабовидящих — подробности

### Высокая контрастность
- Фон: `#000000` (чистый чёрный)
- Текст: `#FFFFFF` (чистый белый)
- Акцент: `#FFFF00` (жёлтый)
- Кнопки: белый текст на чёрном фоне с яркой рамкой `#FFFF00`
- Улучшенная читаемость для пользователей с нарушениями зрения

### Режим для дальтоников
- Используются цвета, различаемые при дейтеранопии (красно-зелёная слепота)
- Основной акцент: `#0077BB` (синий)
- Дополнительный: `#EE7733` (оранжевый)
- Зелёный заменён на `#33BBEE` (голубой)
- Минимальный риск путаницы красного и зелёного

### Крупный шрифт
- Все шрифты увеличены на 30%
- Заголовок: 34 пт → 44 пт
- Подзаголовок: 16 пт → 21 пт
- Основной текст: 14 пт → 18 пт
- Кнопки: 17 пт → 22 пт
- Высокая читаемость без увеличения системного масштаба

## Примечание по зависимостям

Если при запуске вы видите предупреждения в логе:
- `⚠️ Playwright не установлен!` — выполните `pip install playwright && playwright install chromium`
- `⚠️ openpyxl не установлен!` — выполните `pip install openpyxl`

Всё необходимое для работы уведомлений и обоев (`plyer`, `Pillow`) устанавливается вместе с основными зависимостями.
